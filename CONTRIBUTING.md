# Contributing

Thanks for helping maintain `oss-maintainer-pulse`.

## Good first contributions

- Add a small fixture and test for a GitHub issue or PR shape.
- Improve Markdown wording for maintainers.
- Add a focused CLI option such as `--label` or `--author`.
- Improve documentation with real maintainer workflows.

## Development

```bash
npm test
npm run lint
node bin/oss-maintainer-pulse.js --input test/fixtures.json
```

## Pull request expectations

Please include:

- what changed
- why the change helps maintainers
- validation commands run
- screenshots or sample output when output formatting changes

Small PRs are easier to review and merge.
